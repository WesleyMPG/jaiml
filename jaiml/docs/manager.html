JaimlManager¶

class jaiml.JaimlManager(module_name, brain_file=True, templates_dir_name='jinja-templates')¶

Bases: object

This class has methods to register and manage jinja-aiml(jaiml) templates.

JaimlManager will register any function that uses the decorator marker() as a jaiml provider, which means that function follow the pattern decribed in marker().

Parameters

  • brain_file (bool, defauts to True) – If True a file with the output aiml will be generated.

  • templates_dir_name (str, defaults to 'jinja-templates') – path to templates folder.

property brain_file¶
property header¶

The header to be placed on the output and all aiml files.

Defauts to ‘<aiml version=”1.0” encoding=”UTF-8”>\n’.

property footer¶

The footer to be placed on the output and all aiml files.

Defaults to ‘\n</aiml>’.

get_output()¶

Returns the result of joining all jaiml templates.

Return type

str

generate_file(template_name, headerless=True, ext='.aiml')¶

Generates a file with the output of a jaiml provider function.

Parameters

  • template_name (str) – Name of the function to have a file generated.

  • headerless (bool, optional) – If False the generated file will have, defaults to True.

  • ext (str) – output file extension, defaults to ‘.aiml’.

Example:
@jaiml_manager_instance.marker
def foo():
  return 'some_file.xml', vars_dict

jaiml_manager_instance.generate_file('foo')
generate_all(headerless=True, ext='.aiml')¶

Does the same as generate_file() but to all jaiml provider functions.

marker(function)¶

This method is a decorator that marks a function as a jinja-aiml provider.

It must be a function which returs two things:

  • A string buffer containing jinja-aiml (xml) syntax or the name of a file from jinja-templates folder at the root of your project.

  • A dic with keys which the names are the variable names used in your template.

Example 1:
def ex1():
  buffer = """
  <category>
      <pattern>HELLO</pattern>
      <template>
          <random>
          {% for response in responses %}
            <li>{{ response }}</li>
          {% endfor %}
      </random>
      </template>
  </category>
  """
  responses = ['HELLO SIR', 'How Are you?', 'Hi!', 
                'Hello dear', 'Hi, are you ok?']
  vars = {
    'responses': responses,
  }
  return buffer, vars
Example 2:
def ex2():
  responses = ['HELLO SIR', 'How Are you?', 'Hi!', 
                'Hello dear', 'Hi, are you ok?']
  vars = {
    'responses': responses,
  }
  return 'example-file.xml', vars
